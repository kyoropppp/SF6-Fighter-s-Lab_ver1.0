# ストリートファイター6 キャラクターデータベース

## 概要
ストリートファイター6において、キャラクターを使用する際、または対策を立てる際に必要な情報を効率的に確認できるWebベースのデータベースです。

## 機能

### 主要機能
- **キャラクター選択**: 全キャラクターの一覧から直感的に選択
- **対策情報**: 確定反撃、対空対策、弱点の突き方
- **強い行動**: 主力技、連係、起き攻めパターン
- **コンボレシピ**: 基本、状況別、ゲージ使用コンボ
- **検索機能**: キャラクター名での検索
- **レスポンシブデザイン**: PC・タブレット・スマートフォン対応

### 操作方法
- **キャラクター選択**: 左側のサイドバーからキャラクターをクリック
- **タブ切り替え**: 対策・強い行動・コンボのタブで情報切り替え
- **検索**: 上部の検索バーでキャラクター名を入力
- **キーボードショートカット**:
  - `Ctrl+F` / `Cmd+F`: 検索バーにフォーカス
  - `Escape`: 検索をクリア
  - `1`, `2`, `3`: タブ切り替え（対策、強い行動、コンボ）

## 使用方法

### 基本的な使用方法
1. Webブラウザで `index.html` を開く
2. 左側のキャラクターリストから対象キャラクターを選択
3. 上部のタブで「対策」「強い行動」「コンボ」を切り替え
4. 必要な情報を確認

### データの更新
データの更新は `data/` フォルダ内のJSONファイルを直接編集することで行えます。

#### キャラクターの追加
`data/characters.json` に新しいキャラクターを追加：

```json
{
  "id": "new-character",
  "name": "新キャラクター",
  "nameEn": "New Character",
  "icon": "images/characters/new-character.png"
}
```

#### 対策データの追加
`data/counter_strategies.json` に対策情報を追加：

```json
{
  "characterId": "character-id",
  "characterName": "キャラクター名",
  "punishes": [
    {
      "move": "対象技名",
      "punish": "反撃技・方法",
      "description": "詳細説明"
    }
  ],
  "antiAir": [...],
  "weaknesses": [...]
}
```

## 開発

### 開発環境
1. プロジェクトをダウンロード
2. Webサーバーを起動 (推奨: Live Server等)
3. ブラウザで `index.html` にアクセス

### カスタマイズ
- **CSS変数** を使用したテーマカスタマイズ
- **data/** 内のJSONファイルでデータカスタマイズ
- **JavaScript** でのロジックカスタマイズ

## 注意事項

### データ管理
- JSONファイルを直接編集する際は、JSON形式を正しく保つ
- データの変更後はブラウザのキャッシュをクリア
- 定期的なバックアップを推奨

### 画像ファイル
- キャラクターアイコンは `images/characters/` に配置
- 推奨サイズ: 64x64px以上
- 対応形式: PNG, JPG, WebP

### パフォーマンス
- 大量のデータを扱う場合は、データの分割を検討
- 画像ファイルは適切なサイズに最適化
